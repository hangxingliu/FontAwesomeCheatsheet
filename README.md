# FontAwesome Cheatsheet

一个中文版本的[Font Awesome](http://fontawesome.io/)图标库的速查手册

## 使用

1. 把这个项目clone下来
2. 用浏览器打开`cheatsheet.html`
3. `ctrl+f`查找你需要的图标的关键字就能找到对应的图标的样式名了
